IMU production line label printing software code. Automates data retrieval from a Microsoft SQL database server using input data to form fill label templates for station printing.

Runs on Windows 10
Gets data from database
Autofills label template csv
Sends a print command to Bartender

Python libraries:
csv (included)
json (included)
os (included)
pyodbc 4.0.30
shutil (included)
sys (included)
tkinter (included)
traceback (included)
