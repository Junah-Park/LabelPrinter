Development code for label printing software used in IMU production lines.
Automates data retrieval from a Microsoft SQL database server by querying input data through a tkinter GUI
Form fills label templates for station printing.

Runs on Windows 10
Gets data from database
Autofills label template csv
Sends a print command to Bartender

Python libraries:
csv (included)
json (included)
os (included)
pyodbc 4.0.30
shutil (included)
sys (included)
tkinter (included)
traceback (included)
